Extraction rules:

- Extract only actionable TODOs.
- A TODO is actionable when it implies a concrete task, follow-up, reminder, or deliverable.
- Ignore background context, commentary, status, or vague hopes that are not clear actions.
- Prefer shorter normalized titles while keeping the task meaning intact.
- Keep task ordering aligned with source order inside the target block.
- Preserve nesting depth for bullet-like structures when it is visible in the target block.
